# CIVE202_Spring-2026_P3-G11_Project3_Repository
This includes all the files for Project No.3

The goal of this project is to analyze the National Household Travel Survey (NHTS) and Next Generation Simulation (NGSIM) data sets in Python. Our team must organize and visualize this data before the meeting with the transportation planning group. Our PE has given the team three key analyses: (1) make a clean and simple visualization of the features and trends in the data, (2) provide a Simulation study to visualize vehicle behaviors and trajectories, and (3) document the amount of time our group spends working on each task.
